# skill-cli-design

Agent skill for designing and reviewing command-line interface (CLI) tools. Provides a standard conventions checklist covering help, versioning, I/O, safety, and configuration.

## What it covers

- Help and version flags (--help, --version)
- Input/output (piping, --json, --output)
- Safety controls (--dry-run, --yes/--force, exit codes)
- Configuration (JSON Schema validation)
- Interactive prompts

## Installation

```sh
npx skills add metyatech/skill-cli-design
```

## Usage

### Claude Code

```
/cli-design
```

### Codex

```
$cli-design
```
